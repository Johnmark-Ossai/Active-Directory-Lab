# Windows Server 2022 Domain Controller Deployment

## Objective
Deploy a Windows Server 2022 Domain Controller for centralized authentication and management.

## Server Information

- OS: Windows Server 2022
- Domain: lab.local
- Server Role: Domain Controller
- Services: AD DS, DNS, DHCP

## Configuration Steps

1. Install Windows Server 2022
2. Configure Static IP Address
3. Install Active Directory Domain Services
4. Promote Server to Domain Controller
5. Configure DNS
6. Configure DHCP Scope
7. Create Organizational Units
8. Create Users and Security Groups
9. Join Client Computer to Domain

## Outcome

Successfully deployed a functional Active Directory environment supporting centralized user authentication and policy management.
